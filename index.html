2 - WWW (World Wide Web)
- a collection of HTML docs (HTML = hyper text markup language), that glue everything together
- invented in the early 90s
- 30+ billion pages
- links (hyper linkgs) create the web

3 - Major "pieces" of the web:
- you :)
- PC/Comp running a browser
- the internet
- servers

The internet and servers talk to each other through http.

4 - HTML - hyper text markup language, made up of... - text content - what you can see
- markup - what it looks like
- references to other docs (images/videos, etc.)
- links to other pages
- HTML - hyper text markup language, made up of --

HTML markup is made of tags. There is an opening tag, followed by content, and then a closing tag. Each tag is named by a tag description within brackets. An entire tag is called an element.

5 - HTML attributes
- tag attributes ascribe a value
- tags can have multiple values

An example of an attribute assigned to a tag is an anchor. Anchors are for making links, like assigning a URL as an attribute to piece of content.

5a - Image tags
- Unlike other tags, image tags don't have a closing tag. This is called a voide tag, because it has no content.
- Alternate text can be assigned to display when the image doesn't load - this isn't required, but is good coding citizenry.

5b - Whitespace
- All line spaces must be defined, forcing line breaks through tags.
- Void tags like br don't need a close; but paragraph tags do need a close

5c - Inline or block
- inline tags are used to simply end a line of text, and are void tags like line break = br
- block tags make an invisible box for the text, and these can be assigned height and width

5d - Span/Div
- span represents text, and is an inline tag
- div represents text, and is a block tag; it behaves like a paragraph and creates a box around the tagged section.

6 - An HTML Document
- first, doctype specifies the type of document at the beginning
- next is an opening doc tag, like html to indicate the beginning of the doc
- then attributes like head, title, etc., where you would define CSS guidelines
- after the head and title tags, the body of the document is coded
- once completed, a closing html tag is included
